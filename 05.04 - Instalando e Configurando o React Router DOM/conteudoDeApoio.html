Atualização

No dia 09/11/2021 houve uma atualização na major do React Router. Com isso, algumas APIs mudaram.

Recomendamos acessar o código fonte.

Para instalar na versão específica, basta remover o pacote atual (se existir) e rodar o comando abaixo:

[code lang="bash"] yarn add react-router-dom@5.3.0 && yarn add -D @types/react-router-dom@5.3.0 [/code]
